PirateFlix is a small movie discovery site inspired by Rotten Tomatoes and modern streaming UIs. It features a 3D rotating carousel of posters, clickable movie detail pages with scores and trailers.

- **3D Movie Carousel**
  - Rotating carousel of trending movies built with CSS 3D transforms and JavaScript.
  - Each poster is clickable and routes to a dedicated detail page for that movie.

- **Movie Detail Pages**
  - Per‑movie page that shows title, year, runtime, genres, critic score, audience score, and description.
  - Embedded YouTube trailer for each movie.
  - “Where To Watch”, “What To Know”, and “Cast & Crew” sections populated from a shared JavaScript data object.
  - Top hero banner that uses the movie’s poster/backdrop as a background, similar to Rotten Tomatoes.
 
  - **TECHNOLOGIES**
  - CSS for the design and colors.
  - js for the carousel, and clickable text
  - html for the website structure
